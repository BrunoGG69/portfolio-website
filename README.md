# BrunoGG's Portfolio Website

### This is my personal portfolio website. Visit it on https://brunogg.in

## Technologies used:
- HTML
- CSS
- JavaScript
- Tailwind CSS
- Framer Motion
- React

## Features
- Responsive design
- Dark mode
- Smooth animations
- Contact form
- Projects section
- Infinitely Looped icons 
- Custom Cards for Information

## How to run the project locally
1. Clone the repository
2. Run **npm install**
    ```bash
    npm install
    ```
3. Run **npm run dev**
    ```bash
    npm run dev
    ```
